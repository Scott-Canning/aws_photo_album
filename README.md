# aws_photo_album

Cloud Computing assignment designed to allow users to upload and search for images based on AI generated and user provided labeling.

AWS microservices utilized include:
- S3
- API Gateway
- Lambda
- Lex
- OpenSearch (ElasticSearch)
- Rekognition
- CloudWatch
